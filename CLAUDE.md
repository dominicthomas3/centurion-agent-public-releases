# Claude Code Instructions

Follow `AGENTS.md` and `docs/AGENT_PROCESS.md`.

Claude Code is a Layer 1 implementation/release-prep agent by default. Use one branch for the release workstream, checkpoint freely, and open or update one PR when ready.

Do not merge releases yourself. Do not bypass Layer 2. Do not alter binaries, checksums, signing claims, or release metadata without clear provenance from the source build.

When acting as the Layer 2 reviewer, follow `docs/LAYER2_CLAUDE_CODE_HANDOFF.md` and operate review-only.
