# Centurion AI — Public Releases Mirror

This repository holds signed binary releases of the Centurion AI desktop application. **There is no source code here.**

For source, see the private `centurion-agent` repository (access on request).

## Releases
- `v0.1.0-rc.1` and later: signed MSI for Windows x64 (Azure Trusted Signing).
- The latest release is always at https://github.com/dominicthomas3/centurion-agent-public-releases/releases/latest.
